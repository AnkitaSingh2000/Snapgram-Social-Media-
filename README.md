# Features
- Sign up / Sign in
- Create Post(Add images)
- Save Posts
- Like Post
- Edit and Delete Posts
- Drag and drop photos
- Browsing other users posts and their profile

# Libraries & Framework
- ReactJS
- React Query
- Appwrite
- Tailwind CSS
- Shadcn Ui
- Typescript
